我把声音做成产品。

我叫王众，大家一般叫我加菲众。2006 年开始，我在 HIT FM 和香港 URadio 做主持与音乐统筹。后来，我把广播里练出来的判断力带进产品，做出了 PILO 助眠音乐枕头，也为 Nike 上海马拉松做了 Run Baby Run®。

现在我继续沿着这条线往前走，用 AI 扩展声音的表达和使用场景。Talktalk 是我在做的提词系统，帮助主播和创作者更顺畅地表达、组织叙事；同时，我也在探索面向睡眠、专注与表现的自动混音系统。我曾担任第47届 Emmy 奖声音单元评委。

我做过什么

这些作品，比自我介绍更能说明我是谁。

PILO

把夜晚的陪伴感，做成一件经得起市场检验的产品。

做 PILO 最难的，从来都不只是技术实现，而是要同时解决几组彼此冲突的问题：得舒服、得好听、得好看，还得让市场愿意接受。它不能只是一个概念上很动人的东西，只适合放进展览里；它必须是一个你每天晚上真的会躺上去的产品。

后来它拿了 Red Dot、A' Design、金点奖、Plus X、K-Design。对我来说，重点不在“奖很多”，而在于几套背景完全不同的评审体系，最后给出了相近的判断：这东西是想清楚了的。

它进了 Brookstone 中国门店，上了京东，也在 Amazon US 和 Japan 卖过。产品到底站不站得住，真实货架会比任何漂亮描述都更诚实。

Run Baby Run®

把节奏感做成一个真正进入训练与赛事的声音项目。

Run Baby Run® 最早是我跑步时反复琢磨出来的问题：能不能用节奏引导配速，让声音不只是背景，而是训练的一部分。后来它慢慢长成了一个文化 IP，被耐克合作使用，成为上海马拉松的官方专属配乐，也收录进了训练手册。

我更在意的，不只是一次品牌合作，而是一个个人创意从一开始就为真实场景而设计，最后真的进入了品牌和赛事的基础设施里。

现在在做的事

Talktalk，以及一套正在成形的 AI 音频系统。

Talktalk 是一个帮助主播和创作者更顺畅表达、组织叙事的提词系统。它不是从一张功能清单开始，而是从真实表达里那些会让人卡住的时刻开始。

与此同时，我也在探索面向睡眠、专注和表现的自动混音系统。技术底座在变，但我关心的问题没变：怎样让声音更有用、更贴身，也更有效。

我其实擅长什么

我真正擅长的，不是讲一个想法，而是把它落成产品。

我最擅长的，是把模糊的需求做成完整的产品，再把它送到真正需要它的场景里。

01

判断什么值得被听见

做广播时练出来的，是在很短的时间里判断什么会留下、什么会被略过。后来我发现，这种直觉在产品设计里比在节目里更有价值。

02

把感受变成实体

PILO 不是概念图，而是一件能被购买、被触摸、被长期使用的产品。把“安稳”“陪伴”“私密”这些词做成一个实体，远比说出来难。

03

让作品找到归宿

耐克合作、零售渠道、媒体报道，不是做完之后碰上的运气，而是从设计第一天就想进去的结果。产品的命运，最好一开始就写在设计里。

别人如何看这件事

媒体、机构和合作方，给了这件事另一种更外部的判断。

媒体视角

他们看到的，不只是产品本身，还有它为什么成立。

Wired、Men's Health、Cosmopolitan 中国版和《新周刊》的共同点，不是都报道了某个单品，而是都注意到：我做的不是一个孤立的创意，而是一整套被设计出来的使用场景。

公开验证

奖项、渠道和合作，让主观判断变成了可被验证的事实。

对外界来说，这些信号的意义不在装点门面，而在说明这件作品不只在我的叙述里成立，它也经得起评审、零售和真实使用场景。
