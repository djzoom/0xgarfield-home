PILO

把夜晚的陪伴感，做成一件经得起市场检验的产品。

做 PILO 最难的，从来都不只是技术实现，而是要同时解决几组彼此冲突的问题：得舒服、得好听、得好看，还得让市场愿意接受。它不能只是一个概念上很动人的东西，只适合放进展览里；它必须是一个你每天晚上真的会躺上去的产品。

后来它拿了 Red Dot、A' Design、金点奖、Plus X、K-Design。对我来说，重点不在“奖很多”，而在于几套背景完全不同的评审体系，最后给出了相近的判断：这东西是想清楚了的。

它进了 Brookstone 中国门店，上了京东，也在 Amazon US 和 Japan 卖过。产品到底站不站得住，真实货架会比任何漂亮描述都更诚实。

Run Baby Run®

把节奏感做成一个真正进入训练与赛事的声音项目。

Run Baby Run® 最早是我跑步时反复琢磨出来的问题：能不能用节奏引导配速，让声音不只是背景，而是训练的一部分。后来它慢慢长成了一个文化 IP，被耐克合作使用，成为上海马拉松的官方专属配乐，也收录进了训练手册。

我更在意的，不只是一次品牌合作，而是一个个人创意从一开始就为真实场景而设计，最后真的进入了品牌和赛事的基础设施里。

当前——Talktalk

Talktalk：AI-native 音频赛道的第一块楔子。

Talktalk 是面向口语表达"卡壳时刻"的实时工具——为主播、创作者、所有在压力下用语言工作的人。它是一个更广赛道的第一块产品楔子：能感知听者状态、实时塑形音频、围绕"人在做什么"组织声音的系统。

Talktalk 背后是 PILO 与 Run Baby Run® 同一套论点：把声音当作 剂量 来设计，而不是 delivery。下一组应用：专注与运动表现。更多文字和实验会陆续放到这个网站上。
